นี่คือแอปพลิเคชันที่ช่วยให้คุณออกแบบข้อมูล json ได้อย่างรวดเร็วโดยที่คุณไม่มีความรู้เกี่ยวกับซอร์สโค้ดข้อมูล json เลย

แอปพลิเคชันยังมีฟังก์ชันในการดูซอร์สโค้ดที่คอมไพล์แล้วเป็นทรีเพื่อดูวัตถุในสตริง json

สร้างและจัดเก็บสตริง json ในโครงการเพื่อใช้ตามต้องการ

เพิ่มคุณสมบัติ อาร์เรย์ และดัชนีง่ายๆ ตั้งค่าคุณสมบัติอย่างรวดเร็วและเป็นมืออาชีพ
